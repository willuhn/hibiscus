Hibiscus Dokumenation

Hibiscus ist eine Online-Banking Anwendung für das deutsche HBCI Protokoll. Sie basiert auf der HBCI Implementierung HBCI4Java und läft als Plugin innerhalb des Frameworks Jameica.

Support

Bei Problemen mit der Installation oder Einrichtung von Hibiscus schauen Sie einfach in das Forum www.onlinebanking-forum.de.
Sollte Ihnen dort niemand helfen können, dann wenden Sie sich bitte per Mail an hibiscus@willuhn.de. Beachten Sie hierbei bitte folgende Hinweise:
  • Beschreiben Sie den Fehler möglichst exakt - am besten mit dem wortwörtlichen Fehlertext oder einem Screenshot.
  • Geben Sie die Versionsnummer von Hibiscus und Jameica sowie das von Ihnen verwendete Betriebssystem an.
  • Senden Sie nach Möglichkeit die erzeugte Log-Datei "jameica.log" im Anhang mit. Sie finden die Datei in Ihrem Benutzerverzeichnis:
    Windows C:\Dokumente und Einstellungen\<benutzername>\.jameica
    Linux/Linux-AMD64/MacOS /home/<benutzername>/.jameica
    Öffnen Sie die Datei bitte vor dem Senden noch in einem Texteditor und schwärzen Sie ggf. vorhandene persönliche und sensible Daten durch das Zeichen "X" aus.
Falls Sie einen Fehler in Hibiscus gefunden haben oder Sie sich eine Verbesserung wünschen, dann schauen Sie einfach mal in das Bugtracking-System Bugzilla unter http://www.willuhn.de/bugzilla/. Sollte Ihr Fehler oder Wunsch dort noch nichts erfasst sein, dann können Sie sich selbst einen Bugzilla-Account einrichten und einen neuen Bug-Report verfassen. Beachten Sie hierbei ebenfalls die oben stehenden Hinweise. Sie werden automatisch per Mail informiert, wenn der Report bearbeitet wird.
