Update einer existierenden Installation

  1. Erstellen Sie eine Sicherheitskopie Ihrer Hibiscus-Benutzerdaten. Sie befinden sich im Verzeichnis:
    Windows C:\Dokumente und Einstellungen\<benutzername>\.jameica
    Linux/Linux-AMD64 /home/<benutzername>/.jameica
    MacOS .jameica (im Benutzerverzeichnis)
  2. Wechseln Sie nun in das Jameica-Programm-Verzeichnis und löschen es vollständig um sicherzustellen, dass nach dem Update keine alten Programm-Bestandteile vorhanden sind, welche später Fehler auslösen können. Das Programm-Verzeichnis befindet sich in der Regel in:
    Windows C:\Progamme\jameica
    Linux/Linux-AMD64 /opt/jameica
    MacOS Löschen Sie das Programm "Jameica"
    Ihre Benutzerdaten gehen hierbei nicht verloren - sie befinden sich ausschliesslich in dem oben genannten Benutzerverzeichnis.
  3. Fahren Sie nun entsprechend einer Neuinstallation fort. Beim nächsten Start der Anwendung werden Ihre Benutzerdaten automatisch auf die neue Version aktualisiert.
